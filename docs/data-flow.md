## 二、用户使用流程

### 流程 1：学生请求出题

```
学生: "给我出一道Python变量的题目"
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 1. API 层接收请求                                            │
│    POST /api/chat/ {message: "给我出一道Python变量的题目"}   │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. Agent Router 判断请求类型                                 │
│    - 不是简单请求（不是问候/告别）                            │
│    - 路由到 Agent Loop                                       │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. Agent Loop 第 1 轮                                        │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 3.1 加载记忆                                         │  │
│    │     - 短期记忆：无历史                               │  │
│    │     - 长期记忆：掌握度、错误模式                     │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 3.2 构建消息                                         │  │
│    │     - 系统提示：你是一个教育助手...                  │  │
│    │     - 记忆上下文：掌握度50%，薄弱知识点...           │  │
│    │     - 用户消息：给我出一道Python变量的题目           │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 3.3 调用 LLM                                        │  │
│    │     LLM 分析：学生想要练习题，主题是Python变量       │  │
│    │     LLM 决策：调用 generate_question 工具            │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 3.4 解析响应                                         │  │
│    │     {                                                │  │
│    │       "tool_calls": [{                               │  │
│    │         "name": "generate_question",                 │  │
│    │         "arguments": {"topic": "Python变量"}         │  │
│    │       }]                                             │  │
│    │     }                                                │  │
│    └─────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. 执行工具                                                  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 4.1 自动注入 user_id                                 │  │
│    │     arguments: {"topic": "Python变量", "user_id": "..."}│
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 4.2 调用 LLM 生成题目                               │  │
│    │     - 构建提示：请生成一道关于"Python变量"的题目     │  │
│    │     - LLM 返回 JSON 格式的题目                       │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 4.3 保存题目到 MongoDB                               │  │
│    │     - question_id: q_Python变量_medium_选择题        │  │
│    │     - 保存到 questions 集合                          │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 4.4 返回结果                                         │  │
│    │     ToolResult: {success: true, data: {题目数据}}    │  │
│    └─────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 5. Agent Loop 第 2 轮                                        │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 5.1 添加工具结果到消息                               │  │
│    │     - 工具执行成功，返回题目数据                     │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 5.2 调用 LLM                                        │  │
│    │     LLM 分析：题目已生成，需要返回给学生             │  │
│    │     LLM 决策：返回最终答案                           │  │
│    └─────────────────────────────────────────────────────┘  │
│    ┌─────────────────────────────────────────────────────┐  │
│    │ 5.3 解析响应                                         │  │
│    │     {                                                │  │
│    │       "final_answer": "好的，这是一道关于Python变量..."│
│    │     }                                                │  │
│    └─────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 6. 返回响应给学生                                            │
│    {                                                         │
│      "response": "好的，这是一道关于Python变量的题目...",     │
│      "tool_calls": [{"name": "generate_question", ...}],     │
│      "iterations": 2                                         │
│    }                                                         │
└─────────────────────────────────────────────────────────────┘
```

### 流程 2：学生提交答案（答错）

```
学生: "答案是A"
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 1. Agent Loop 第 1 轮                                        │
│    LLM 分析：学生提交答案，需要检查                          │
│    LLM 决策：调用 check_answer 工具                          │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. 执行 check_answer 工具                                    │
│    - 从 MongoDB 获取题目                                     │
│    - 比较答案：A ≠ C（正确答案）                             │
│    - 返回：{is_correct: false, correct_answer: "C"}          │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. Agent Loop 第 2 轮                                        │
│    LLM 分析：答案错误，需要记录错题                          │
│    LLM 决策：调用 record_error + update_mastery 工具         │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. 执行 record_error 工具                                    │
│    - 调用 LLM 分析错误类型（混淆/粗心/概念/遗忘）           │
│    - 保存到错题本（MongoDB）                                 │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 5. 执行 update_mastery 工具                                  │
│    - 更新掌握度：Python变量 50% → 35%                        │
│    - 保存到长期记忆（MongoDB）                               │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 6. Agent Loop 第 3 轮                                        │
│    LLM 分析：已记录错题，可以返回反馈                        │
│    LLM 决策：返回最终答案                                    │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│ 7. 返回响应给学生                                            │
│    "❌ 回答错误。正确答案是：C                               │
│     解析：变量本质上是对内存中对象的引用...                  │
│     📝 已自动记录到错题本"                                   │
└─────────────────────────────────────────────────────────────┘
```

│   ├── created_at

│   ├── question_id
│   ├── topic
│   ├── difficulty
│   ├── question_type
│   ├── question
│   ├── options[]

│
├── learning_states          # 学习状态表

    ├── content
    └── difficulty

    ├── topic
    ├── difficulty

## 四、关键数据流

### 1. 题目生成数据流

```
LLM 生成题目
    │
    ▼
解析 JSON
    │
    ▼
添加 question_id
    │
    ▼
保存到 MongoDB (questions)
    │
    ▼
返回给学生
```

### 2. 答案检查数据流

```
学生提交答案
    │
    ▼
从 MongoDB 获取题目
    │
    ▼
比较答案
    │
    ├─ 正确 → 更新掌握度 (+10%)
    │
    └─ 错误 → 记录错题 + 更新掌握度 (-15%)
              │
              ▼
         LLM 分析错误类型
              │
              ▼
         保存到错题本
```

### 3. 记忆注入数据流

```
每次 LLM 调用前
    │
    ▼
加载长期记忆
    │
    ▼
构建记忆上下文
    │
    ▼
注入到系统提示
    │
    ▼
LLM 调用
```
